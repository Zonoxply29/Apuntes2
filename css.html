Propiedades de CSS

Propiedades para texto

  • font-size altera el tamaño del texto y se define en pixeles
  • font-familyagrega una tipografia al texto
  • text-aligncuenta con varias propiedades para alinear el texto en diferentes formas,la mas usada es center
  • text-decorationAgrega un formato en el texto, las tres mas usadas son line-throught, underline, none
  • font-weightEspecifica el grueso de la letra, se recomienda ponerle 600 u 800 NO LLEVA PIXELES
  • font-stylePermite cambiar entre los estilos de letra Italic,Normal,Oblicual
  • colorDa color al texto acepta numeros hexadecimal o colores escritos

Propiedades background

  • background-colorPermite agregar un color al fondo acepta numeros hexadecimal o colores escritos
  • background-imagePermite agregar una imagen al fondo con una ruta "url('');"
  • background-positionDefine la posicion inicial de el fondo
  • background-repeatEl mas utilizado es no-repeat, esto debido a que el navegador rellena los espacios vacios repitiendo la imagen tanto en vertical como horizontal, por eso existen tambien las propiedades repeat-x , repeat-y para que NO repitan la imagen en el eje x o el eje y

Propiedades para tamaños de contenedores

  • height Define la altura de el contenedor y se puede ocupar px, vh o porcentajes
  • width Define el ancho de el contender y se puede ocupar px, vh o porcentajes
