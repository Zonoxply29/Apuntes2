• git init- inicar un proyecto
  • git status - Estatus del proyecto
  • git add -A - agregas todos los archivos a la zona de trabajo (Stage Area)
  • git commit -m "" - Agregas un comentario sobre lo que hiciste en los archivos
  • git branch NombreDeLaRama - Crea una Rama
  • git branch - Muestra una lista de todas las rama que tengo
  • git checkout NombreDeLaRama - Te mueve a el nombre de la rama que pongas
  • git merge NombreDeLaRama - Fusiona las ramas
